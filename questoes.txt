1.O que é um repositório no Git e qual a sua importância?
    Um repositório no Git, serve como um local para salvar e compartilhar cada fase dos projetos feitos. É importante para registrar as alterações em um projeto, podendo voltar em "versões" anteriores. 
2.Qual a função do Git Bash?
    O git bash nos permite versionar os projetos, interpretando os comandos para o Git.
3.O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
    O comado git add serve para adicionar os arquivos para commit, "<arquivo>" adiciona um arquivo específico, "add ." adiciona todo o conteúdo dentro da pasta.
4.Por que utilizamos o ponto (.) no comando git add .?
    O ponto refere-se a todo o conteúdo que está na pasta.
5.Explique o comando git commit e por que é importante criar uma mensagem descritiva.
    Serve para registrar as adições feitas, a mensagem é importante para a descrição do que está sendo feito no commit, assim podendo entender melhor as mudanças feitas.
6.Qual a função do comando git push?
    Tem a função de enviar o último commit feito para o repositório.
7.Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
    É importante para salvar cada alteração feita no projeto, caso necessário permitindo voltar a versões anteriores.  
8.Explique a função das tags HTML e como elas estruturam uma página.
